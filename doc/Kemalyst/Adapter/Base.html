abstract class Kemalyst::Adapter::Base

  • Kemalyst::Adapter::Base
  • Reference
  • Object

Overview

The Base Adapter specifies the interface that will be used by the model objects to perform actions against a specific database. Each adapter needs to implement these methods.

Direct Known Subclasses

  • Kemalyst::Adapter::Mysql
  • Kemalyst::Adapter::Pg
  • Kemalyst::Adapter::Sqlite

Instance Method Summary

  • #clear(table_name)

    clear will remove all rows from a table and reset the counter on the id.

  • #create(table_name, fields)

    create will create the table based on the fields specified in the sql_mapping defined in the model.

  • #delete(table_name, id)

    This will delete a row from the database.

  • #drop(table_name)

    drop will drop the table

  • #env(value)

    method used to lookup the environment variable if exists

  • #insert(table_name, fields, params) : Int64

    This will insert a row in the database and return the id generated.

  • #select(table_name, fields, clause = "", params = {} of String => String)

    select performs a query against a table.

  • #select_one(table_name, fields, id)

    select_one is used by the find method.

  • #type_mapping(db_type)

    method to perform a reverse mapping of Database Type to Crystal Type.

  • #update(table_name, fields, id, params)

    This will update a row in the database.
