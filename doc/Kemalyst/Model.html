def self.all(clause = "", params = {} of String => String) #

All will return all rows in the database. The clause allows you to specify a WHERE, JOIN, GROUP BY, ORDER BY and any other SQL92 compatible query to your table. The results will be an array of instantiated instances of your Model class. This allows you to take full advantage of the database that you are using so you are not restricted or dummied down to support a DSL.

macro sql_mapping(fields, table_name = nil, timestamps = true) #

sql_mapping is the mapping between columns in your database and the fields in this model. proerties will be created for each field. The type of the field is specific to the database you are using. You may specify other criteria for each field like NOT NULL and Referential Integrity. This allows you to take full advantage of the database of choice. you may also specify a specific table_name and if you want the timestamps or not. This will help with backward compatibility of existing databases.
