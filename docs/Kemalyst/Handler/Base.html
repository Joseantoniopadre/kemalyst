class Kemalyst::Handler::Base

  • Kemalyst::Handler::Base
  • Reference
  • Object

Overview

The base class for Kemalyst handlers. This extension provides a singleton method and ability to configure each handler. All configurations should be maintained in the /config folder for consistency.
